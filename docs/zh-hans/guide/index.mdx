# 初次相见

欢迎使用 Palpo, 一个社区开发和维护的 Matrix 服务器，准循 Apache 2.0 开源协议。

## 为什么创建这个框架

作为一个初学者，我发现自己很难掌握像 actix-web 和 Rocket 这样的现有框架。当我尝试用 Rust 重写一个 Go Web 服务时，每个框架似乎都比 Go 中的框架更复杂。Rust 已经有陡峭的学习曲线——为什么还要让 Web 框架变得更加复杂呢？

当 Tokio 发布 Axum 时，我非常激动，以为终于可以停止维护自己的 Web 框架了。然而，在实践中，Axum 表面上的简单掩盖了过度的类型体操和泛型定义。即使编写一个简单的中间件也需要深厚的 Rust 专业知识和冗长、晦涩的样板代码。

因此，我决定继续维护自己的框架——一个直观、功能丰富且对初学者友好的框架。

## Palpo 适合你吗？

Matrix 协议是先进的去中心化的加密通讯协议，它能实现端到端加密，确保你的信息不会在传输中被窃取。如果你的需求符合下面的描述，那么 Palpo 将非常适合你。

- 你关注个人通讯隐私;
- 你希望控制个人聊天数据，不希望他们被大公司分析和非法使用;
- 你不希望重要的商业数据在通讯过程中被窃听.
