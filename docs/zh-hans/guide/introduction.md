# Palpo

Palpo 是一个高性能、可扩展的 Matrix Homeserver 实现。

## 什么是 Matrix？

Matrix 是一个用于安全、去中心化实时通信的开放标准和协议。它提供了一个联邦化网络，不同服务器上的用户可以无缝地进行聊天、语音和视频通话以及文件共享，同时具备强大的端到端加密保障。你可以将它理解为私有化即时通讯平台的开放替代方案。

Matrix 协议的核心特性：

- **去中心化** — 没有单一的控制点，任何人都可以运行自己的服务器，并与更广泛的网络互通。
- **端到端加密** — 私密对话始终保持私密，由 Olm/Megolm 加密协议提供安全保障。
- **互操作性** — 通过桥接功能，Matrix 用户可以与 Slack、Discord、IRC 等平台上的用户互相通信。

如需深入了解 Matrix 协议，请访问 [matrix.org](https://matrix.org/)。

## 为什么选择 Palpo？

- **采用 Rust 构建** — Palpo 使用 Rust 编写，提供内存安全、高并发和出色的运行时性能，且无需垃圾回收器的额外开销。
- **高性能** — 旨在以最少的资源消耗处理大量的用户和消息。
- **可扩展** — 模块化架构使其易于扩展和添加新功能，以适应不断增长的需求。
- **安全** — 全面支持端到端加密，确保您的通信安全。
- **易于部署** — 提供多种部署选项，包括 Docker、Kubernetes 和独立二进制文件。

## Palpo 适合您吗？

如果您有以下需求，Palpo 将是一个不错的选择：

- 一个轻量、资源高效的 Matrix Homeserver，用于个人使用或中小型社区。
- 一个基于 Rust、具有出色性能特性的服务器。
- 简单的部署流程和较低的运维开销。

如果您需要一个久经考验、生态支持最广泛的服务器，也可以考虑 [Synapse](https://github.com/element-hq/synapse)（Python 编写的参考实现）或 [Dendrite](https://github.com/element-hq/dendrite)（Go 编写）。
