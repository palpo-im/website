# Palpo

Palpo 是一个用 Rust 编写的高性能、可扩展的 Matrix Homeserver 实现。

如果你是 Palpo 的新用户，请先阅读[介绍](./introduction.md)了解项目和 Matrix 协议，或直接跳转到[快速开始](./quick-start.md)指南来部署你的服务器。
