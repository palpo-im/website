# Palpo

Palpo is a high-performance, scalable implementation of a Matrix Homeserver written in Rust.

If you are new to Palpo, start with the [Introduction](./introduction.md) to learn about the project and the Matrix protocol, or jump directly to the [Quick Start](./quick-start.md) guide to set up your server.
